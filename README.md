# 🎬 KYLE'S SAGA — Cinematic Anime Film Project

A complete AI-generated dark fantasy anime short film production bible.

## 📦 Download

The complete project is in **`film_project.zip`** (~84 MB).

**Direct download:** [film_project.zip](https://github.com/Aron-dem/New-film/raw/main/film_project.zip)

## 📂 What's Inside

```
film_project/
├── index.html       ← Open this in a browser
├── data.js          ← All scenes, prompts, dialogue data
├── script.txt       ← Plain-text full script
├── README.md        ← Project documentation
└── images/          ← 33 images (30 shots + 3 character refs)
```

## 🎭 Features

- ✅ 15 scenes × 2 shots = 30 cinematic widescreen 16:9 stills
- ✅ Image prompts + Video generation prompts for each shot
- ✅ Full English script with timing & emotion cues
- ✅ Voice direction for 4 characters (Kyle, Vahlrok, Nyxara, Narrator)
- ✅ Director's production notes & enhancement ideas
- ✅ Interactive HTML interface with copy-to-clipboard prompts

## 🚀 How to Use

1. Download the ZIP
2. Extract it
3. Open `film_project/index.html` in your browser
4. Browse 5 tabs: Overview, Characters, Scenes & Prompts, Script, Director's Notes
5. Copy any prompt and paste into Veo 3 / Runway / Kling / Sora to generate the video clip
